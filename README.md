How to Use
Enter a WooCommerce store URL (demo URL pre-filled)
Select your export format (CSV, Excel, or JSON)
Choose scraping options (all pages, custom fields)
Click "Scrape Products" to start
Download your extracted data when complete
This web application provides all the functionality of the browser extension in a convenient web interface that requires no installation. The design follows modern UI principles with a purple color scheme that matches the original branding.
